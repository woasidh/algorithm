## python 자료구조 정리

### Stack
#### 기본적으로 배열을 사용함 왼쪽에서 오른쪽으로 쌓아감
- 선언
```python
stack = [];
```

- 넣기
```python
stack.append(1);
```

- 빼기
```python
stack.pop();
```

- peek
```python
stack[-1]
```

### queue
#### 기본적으로 배열을 사용함 오른쪽에서 들어가고 왼쪽에서 나감
#### collections 모듈에서 deque 사용
- 선언
```python
from collections import deque
queue = deque([]);
```

- 넣기
```python
queue.append(1);
```

- 빼기
```python
queue.popleft();
```
